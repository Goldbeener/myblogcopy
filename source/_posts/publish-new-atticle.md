使用hexo新建一篇文章，命令行
``` bash
    hexo new 'article-name'
```
这样就在source/_posts目录下新建了一个以'article-name'为标题的md文档，就在这里面进行文章书写。  

<!-- more -->

### 给文章添加标签           

每一篇文章的md文档开头都可以来对文章进行分类和添加标签           
``` 
title: 标题    

date: 日期

tags: 标签，如果添加多个标签，此处是一个数组
```
